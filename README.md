# 机械加工管理系统

一个轻量、离线的机械加工厂业务管理工具。原料入库 → 加工进度 → 发货管理 → 库存总览，全流程覆盖。支持 CSV 文件存储和浏览器本地存储。

## 技术栈

- **前端框架**：Vue 3（Global build，无需构建工具）
- **存储层**：localStorage / 本地 CSV（FSA API）
- **CSV 解析**：PapaParse
- **零外部依赖**，离线双击即用

## 快速启动

直接双击 `index.html` 用浏览器打开即可使用，无需任何服务端环境。

**启用数据文件夹绑定（推荐）**：
首次打开后，在「数据管理」页点击「绑定数据文件夹」，选择项目根目录下的 `data/` 文件夹。此后数据将自动读写 CSV 文件，刷新不丢失。

## 业务流程

```
原料入库 ──→ 加工进度 ──→ 发货管理 ──→ 库存总览
  （采购材料）  （关联入库单）  （关联加工单）  （自动统计）
```

### 1️⃣ 原料入库
记录采购的钢材等物料信息，系统自动生成入库单号（`RK-YYYYMMDD-NNN`）。

| 字段 | 说明 |
|---|---|
| 入库单号 | 自动生成，如 `RK-20250625-001` |
| 物料名称 | 如 Q235钢板、45#圆钢 |
| 物料类别 | 钢板/无缝管/矩形管/圆钢/冷拉钢/槽钢/其他 |
| 规格 | 如 10mm×1200×2400、Φ50 |
| 数量 / 单价 / 总价 | 总价自动计算 |
| 供应商 | 如 上海宝钢、沙钢集团 |
| 入库日期 | 默认当天 |

### 2️⃣ 加工进度
基于入库原料创建加工单，追踪产品加工状态。

- 关联已有入库单号，追溯每批原料的用途
- 加工单号自动生成（`JG-YYYYMMDD-NNN`）
- 状态流转：待加工 → 加工中 → 检验中 → 已完成 → 已发货
- 完成时可自动填入完成日期

### 3️⃣ 发货管理
基于加工单进行发货，自动关联产品名称和客户。

- 选择加工单号后自动带出产品名称和客户
- 发货单号自动生成（`FH-YYYYMMDD-NNN`）
- 记录发货数量、物流单号

### 4️⃣ 库存总览
自动计算汇总，**只读视图**。

**原料库存**：按物料名称+规格聚合
```
入库量 − 领料量（加工消耗）= 剩余库存
```

**成品待发**：按加工单号聚合
```
加工量 − 已发货量 = 待发数量
```

支持设置**预警线**，剩余库存低于阈值自动标红"需补货"。

### 5️⃣ 数据管理
集中管理所有数据的导入、导出和存储。

- **导出 CSV**：单表或全部导出，UTF-8 BOM 编码，Excel 打开不乱码
- **导入 CSV**：覆盖导入，表头需与导出格式一致
- **绑定文件夹**：启用 File System Access API，数据自动读写到 CSV 文件
- **加载示例数据**：首次使用可快速体验全流程
- **清空数据**：双重确认，防止误操作

## 存储模式

| 模式 | 触发条件 | 说明 |
|---|---|---|
| **FSA 模式** | 用户主动绑定文件夹 | 双向读写 CSV 文件，数据持久化 |
| **浏览器模式** | 默认（离线双击打开时） | 数据保存在浏览器 localStorage |

## 单号格式

所有单号统一格式：`前缀-YYYYMMDD-NNN`（15 字符）

| 前缀 | 模块 | 示例 |
|---|---|---|
| RK | 原料入库 | `RK-20250625-001` |
| JG | 加工进度 | `JG-20250625-001` |
| FH | 发货管理 | `FH-20250625-001` |

## 数据安全

- 数据默认保存在浏览器 localStorage 中
- 绑定文件夹后自动同步 CSV 文件，可用 Excel 直接打开编辑
- CSV 编码为 UTF-8 BOM，中文兼容
- 建议定期「导出 CSV」备份数据

## 系统要求

- **浏览器**：Chrome / Edge 86+（支持 FSA API 以获得最佳体验）
- 离线使用：直接双击 `index.html` 即可
